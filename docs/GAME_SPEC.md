# 게임 명세 (Game Specification)

## 1. 게임 개요

**2048**은 4×4 격자판에서 숫자 타일을 슬라이드하여 같은 숫자를 합치는 퍼즐 게임이다.  
원작: Gabriele Cirulli (2014), MIT License 오픈소스.

---

## 2. 보드 (Board)

- 크기: **4 × 4** (16칸)
- 각 칸은 **빈칸** 또는 **2의 거듭제곱 숫자** (2, 4, 8, 16 … 2048 …)를 가진다.

---

## 3. 게임 시작

1. 빈 4×4 보드 생성.
2. **랜덤한 2개의 빈 칸**에 타일 배치.
   - 타일 값: **2** (90% 확률) 또는 **4** (10% 확률).

---

## 4. 이동 (Move)

플레이어는 **상 / 하 / 좌 / 우** 4방향 중 하나로 슬라이드 입력을 한다.

### 4.1 슬라이드 규칙

이동 방향 기준으로 각 행(또는 열)을 독립적으로 처리한다.

1. **빈칸 제거** — 숫자 타일을 이동 방향 앞쪽으로 모두 밀어붙인다.
2. **병합(Merge)** — 인접한 동일 숫자 두 개를 하나로 합쳐 두 배 값의 타일로 만든다.
   - 한 번의 이동에서 타일 하나는 **최대 1번만** 병합될 수 있다.
3. **다시 빈칸 제거** — 병합 후 남은 빈칸을 채운다.

### 4.2 예시 (왼쪽 이동)

```
이동 전:  [2][ ][2][4]
이동 후:  [4][4][ ][ ]
```

```
이동 전:  [2][2][2][2]
이동 후:  [4][4][ ][ ]   ← 앞의 두 쌍이 각각 병합
```

```
이동 전:  [2][2][4][ ]
이동 후:  [4][4][ ][ ]
```

### 4.3 유효한 이동

이동 후 **보드 상태가 변화했을 때**만 유효한 이동으로 간주한다.  
보드가 변하지 않으면 새 타일을 생성하지 않는다.

---

## 5. 새 타일 생성

유효한 이동 후:
- 남은 **빈 칸 중 하나**를 랜덤 선택.
- 값: **2** (90%) 또는 **4** (10%) 배치.

---

## 6. 점수 (Score)

- 타일이 **병합될 때마다** 병합된 값을 현재 점수에 더한다.
- 예: 2+2 병합 → **+4점**, 512+512 병합 → **+1024점**
- **최고 점수(Best Score)**: 로컬 스토리지에 저장, 세션 간 유지.

---

## 7. 승리 조건

- 보드에 **2048** 타일이 처음 나타나면 **"You Win!"** 메시지 표시.
- 승리 후에도 계속 플레이 가능 (Continue 선택 시).

---

## 8. 패배 조건 (Game Over)

다음 조건을 **모두** 만족하면 게임 오버:

1. 보드에 **빈 칸이 없다**.
2. 인접한 동일 숫자 타일이 **하나도 없다** (어떤 방향으로도 병합 불가).

## 10. 입력 방식

| 플랫폼 | 입력 |
|--------|------|
| 웹 (PC) | 키보드 방향키 (↑ ↓ ← →) |
| 웹 (모바일) | 터치 스와이프 |
| 네이티브 앱 | 터치 스와이프 (PanResponder / Gesture Handler) |

---

## 11. 기타 기능

- **New Game 버튼**: 언제든지 게임 리셋.
- **최고 점수 표시**: 상단 ScoreBar에 항상 노출.
- **애니메이션**: 타일 이동 슬라이드 + 병합 시 팝(scale) 효과.
